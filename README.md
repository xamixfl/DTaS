# DTaS
Laboratory works for the Data Types and Data Structures course at BMSTU  (IU7)

# Типы и структуры данных

Лабораторные работы по дисциплине **«Типы и структуры данных»**, выполненные в рамках обучения на кафедре **ИУ7 «Программное обеспечение ЭВМ и информационные технологии»** МГТУ им. Н. Э. Баумана.

Все проекты написаны на языке **C** и посвящены изучению классических структур данных, алгоритмов их обработки и работе с динамической памятью. Курс ориентирован на формирование навыков проектирования эффективных структур данных и реализации алгоритмов без использования сторонних библиотек. 

---

## Темы лабораторных работ

В репозитории представлены реализации различных структур данных и алгоритмов их обработки, включая:

- линейные структуры данных;
- динамические структуры данных;
- списки;
- очереди и стеки;
- деревья;
- алгоритмы поиска;
- алгоритмы сортировки;
- работу с указателями и динамической памятью.

---

## Используемые технологии

- C
- GCC
- Make
- Linux

---

## Полученные навыки

В ходе выполнения лабораторных работ были получены практические навыки:

- реализации структур данных "с нуля";
- работы с динамической памятью;
- использования указателей;
- анализа сложности алгоритмов;
- проектирования модульных программ на языке C;
- выбора подходящих структур данных для различных задач.

---

## Назначение репозитория

Репозиторий создан исключительно в образовательных целях и содержит лабораторные работы, выполненные в рамках университетского курса **«Типы и структуры данных»** МГТУ им. Н. Э. Баумана.
